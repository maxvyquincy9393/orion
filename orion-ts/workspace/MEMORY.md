# Orion Memory Vault

This file stores pinned long-term facts only.

## Memory Layers

- Vault: `MEMORY.md` (stable, user-important, no auto-decay)
- Episodic: `workspace/memory/YYYY-MM-DD.md` (interaction highlights)
- Semantic: vector memory for retrieval and linking
- Procedural: learned workflows and stable strategies

## Vault Entry Criteria

All criteria must be true:
1. High confidence
2. Stable over time
3. Reuse value for future interactions
4. User-approved or clearly mission-critical

## Entry Format

`- [YYYY-MM-DD] <fact>`
